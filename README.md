# Hotel-Website-MobileView

## Project Description
This is a responsive landing page, for a mobile view hotel booking and accommodation website. The site features a prominent hero banner for apartment searching, showcases special offers, highlights customer reviews, and provides recent blog posts. The primary goal is to provide a clean, intuitive, and highly functional experience for users viewing on smartphones.

## Installation
This is a static website built with HTML CSS and JS. To view it, simply open the index.html file in your web browser. For the best experience, view it using a browser's mobile device simulator.

***

## Usage
The Hotel Website showcases different accommodation options and services. Users can utilize the search bar in the hero section, browse special offers, read customer reviews, and view recent posts from the hotel blog.
